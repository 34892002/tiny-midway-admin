### 本地开发

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### 开发用-数据库命令
```bash
$ npx prisma studio
$ npx prisma migrate dev --name init
$ npx prisma generate
$ npx prisma db pull
$ npx prisma db push
```
